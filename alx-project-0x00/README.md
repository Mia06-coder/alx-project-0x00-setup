# ALX Project 0x00

## Project Setup

This project was bootstrapped using the `create-next-app` CLI tool with the following settings:

- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ ESLint
- ✅ Import alias: `@/*`
- ❌ No `/src` directory
- ❌ No App Router (used Pages Router)

## Getting Started

To run the development server on port 3000:

```bash
npm run dev -- -p 3000
```
